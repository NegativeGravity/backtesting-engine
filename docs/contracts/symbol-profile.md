# Symbol Profile Contract

The symbol profile captures broker-specific instrument rules required for MT5-compatible calculations.

## Price Grid

- Digits
- Point
- Trade tick size
- Tick value

## Volume Grid

- Minimum volume
- Maximum volume
- Volume step

## Contract and Margin

- Contract size
- Calculation mode
- Profit currency
- Margin currency
- Initial and maintenance margin fields

A profile is versioned because broker symbol properties can change.
